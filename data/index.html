What is a firmware update?

Firmware is the software that runs on your ESP32 board — it's what reads the sensors, runs the web dashboard you're looking at right now, and talks to the network. Updating it replaces that software with a newer version, without needing to plug the board into a computer.

How it works, step by step:

  1. Someone builds a new version of this project's code and produces a single file ending in .bin — that file is the new firmware.
  2. You select that .bin file above and press Update Firmware.
  3. Your browser sends the file to the ESP32 over your local Wi-Fi network (not the internet).
  4. The ESP32 writes it into a spare flash memory slot while the old firmware keeps running, so a dropped connection mid-upload doesn't brick the device.
  5. Once the file finishes transferring and checks out, the board automatically restarts and boots into the new firmware. Your saved Wi-Fi, Firebase, pin, and calibration settings are kept — only the code changes, not your configuration.
  6. The dashboard will briefly disconnect during the restart and reconnect on its own a few seconds later.

Keep this browser tab open and stay on the same Wi-Fi network until the upload finishes — closing the tab or losing Wi-Fi mid-upload can interrupt the transfer.
